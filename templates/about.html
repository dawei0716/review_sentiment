Data Set used: 900 positively tagged movie reviews and 900 negatively tagged movie reviews.

Model used: self-implemented (no libraries like sklearn) Naive Bayes model with Laplace smoothing applied.

Accuracy Achieved: 84.5% accuarcy on 200 test sets (100 positive, 100 negative)

Input string is processed using NLTK.

What is "Score"?: The score in itself doesn't have any meaning. It is simply a measure used to determine positive and negative reviews. score > 0 is a positive review. Higher score means the review is more likely to be positive.

See Github for details on how it was implemented.
